## 🌟 Featured Project: Sentinel

**[Sentinel](https://github.com/Ali-Aoun5/sentinel)** — *Founder & Principal Architect*  
A conflict misinformation early warning system that monitors 20 global news sources in real-time. It detects when credible outlets report contradictory facts about the same event, delivering structured intelligence reports with risk scoring, narrative analysis, and journalist verification actions.

- **Impact:** Built solo in two weeks. Deployed end-to-end on free infrastructure. Won two international awards.
- **Tech Stack:** `Apify`, `Qdrant`, `Featherless AI`, `FastAPI`, `HuggingFace Spaces`, `React`, `D3.js`, `sentence-transformers`
- **Read the Writeup:** [Qdrant Blog: GenAI Zürich Winners](https://qdrant.tech/blog/gen-ai-zurich-winners)
